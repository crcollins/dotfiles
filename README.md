Various dotfiles

To install run:

    bash setup.sh

To uninstall run:

    bash setup uninstall

To do a dry run of the install/uninstall add -d.


To install Vundle for vim, run:

    git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim

And then in vim, run:

    :PluginInstall
